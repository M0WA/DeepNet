developer resources

If you consider helping to develop DeepNet, please take the time and to contact us before you start to make sure we can include your changes.

DeepNet is mainly written in C++. It is separated into a set of statically linked libraries and a few executables. The C++ code is documented using doxygen so it should be easy to get into the source code. It is developed using eclipse IDE which I encourage you to use when compiling and developing DeepNet.

There is a doxygen documentation available online.
